Insertion Sort

I'll describe quickly how Insertion-Sort works . This is one of the easiest sorting algorithm and it is very efficient for little amount of data.

It receives an array in input and basically it will sort it by traversing the elements from the 2nd position till the end.
Every time the element is smaller than the one on its left, it is moved backward till it reaches the right position.
